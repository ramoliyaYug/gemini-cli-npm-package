# 🤖 Chat CLI Gemini AI

A lightweight Gemini-powered AI chat for your terminal.

No API key. No setup. Just install and chat.

## Installation

```bash
npm install -g chat-cli-gemini-ai
```

## Usage

```bash
chat --ai
```

Example:

```text
🤖 Gemini Chat
Type 'exit' to quit.

You > Hello
AI  > Hi! How can I help you today?
```

## Features

- 💬 Interactive terminal chat
- ⚡ Global CLI command
- 🔑 No API key required
- 🚀 Zero configuration
- 🖥️ Simple and lightweight
